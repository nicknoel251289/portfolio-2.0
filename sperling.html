ABOUT

Sperling Interactive is a company that focuses on web development, print, branding, search engine optimization, photography, and online marketing. They specialize in certain industry verticles like law and education but are capable of working with any professional market.

OBJECTIVE

Primary goals of the project were:

  • Practice using vanilla JavaScript
  • Implementing CSS grid and Flexbox
  • Visually and mentally, create a more comprehensive design layout
  • Update the sites appearance with a modern edgy feel using the same color scheme
  • Practice using Adobe XD

PROJECT INFO

As a company that has sparked more creativity out of me, I took pleasure on my own, to recreate and develope (still working on) a landing page based around their company. It gave me the opportunity to work with Adobe XD and a chance to hone in on vanilla JavaScript, as I wanted to really impliment more transitions and interactivity to the page.

Again, this project was done for fun and at my own will. What you see are my own interpretations and ideas.

What I learned

This was a great project to spend more time focusing on the basics of HTML5 and CSS3. When I finally started to write the markup, I wanted to take my time to really try and learned to use less but more efficient code. I was able to really understand how both CSS grid and Flexbox are used to compliment each other.

During this project, I also took time to learn a few more aspects of JavaScript with event listeners. Like CSS grid and Flexbox, I learned how the requestAnimationFrame method compliments event listeners very well.
