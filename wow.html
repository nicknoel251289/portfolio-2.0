W.o.W Front-end Web Project

Specifically created for the famous MMORPG World of Warcraft. This is a class picker for the indecisive and a mini front-end project.

VIEW PROJECT

ABOUT

Massively multiplayer online role-playing. Mode(s) Multiplayer. World of Warcraft (WoW) is a massively multiplayer online role-playing game (MMORPG) released in 2004 by Blizzard Entertainment. It is the fourth released game set in the Warcraft fantasy universe.

OBJECTIVE

Primary goals of the project were:

  • Practice using vanilla JavaScript
  • Practice using arrays and array methods
  • Creating objects and sorting through them via loops

PROJECT INFO

With the coming release of Battle for Azeroth, the next upcoming expansion for the game World of Warcraft, most players are at a point where they are trying to figure out what class they want to have as a main character. What I did for fun was create a class picker questionnaire.

The reason for choosing this project is because it combines both my interests in World of Warcraft and would allow me to practice using JavaScript, a language I desire to be extremely fluent in.

What I learned

One of the hardest aspects of this project was the attention to detail and the amount of attention needed. Althought, I was able to complete this project within a couple of days, it took a bit of self-advocacy and reliance to really get a grasp on how things were working. Overall, I was able to complete the project and with a better grasp of the methods and loops used.
